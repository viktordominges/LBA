HTML exercices 01 - 05

Pour écrire un titre principal, il faut utiliser la balise <h1> Exemple : <h1>Titre principal</h1> Pour écrire un sous-titres, il faut utiliser la balise <h2>,<h3>,<h4>,<h5>... Exemple : <h2>Sous-titre</h2> Pour écrire du texte, il faut utiliser la balise <p> Exemple : <p>Texte</p> Pour écrire une liste, il faut utiliser la balise <ul> Exemple : <ul><li>Texte</li></ul> Pour écrire une liste non ordonnée, il faut utiliser la balise <ol> Exemple : <ol><li>Texte</li></ol> Pour écrire une image, il faut utiliser la balise <img> Exemple : <img src="https://via.placeholder.com/150">



Pour écrire un lien, il faut utiliser la balise <a> Exemple : <a href="https://google.com">Google</a>
